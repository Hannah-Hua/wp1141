# K-pop 周邊商品購買平台

這是一個基於 React + TypeScript + Material-UI 的 K-pop 周邊商品購買平台前端應用程式。

## 功能特色

- 🎨 **精緻的 UI 設計**：使用 Material-UI 打造現代化的使用者介面
- 📱 **響應式佈局**：支援各種螢幕尺寸
- 🔍 **商品篩選**：可依娛樂公司、團體、分類進行篩選
- 📊 **商品排序**：支援多種排序方式（新商品、價格、人氣等）
- 🛍️ **商品瀏覽**：精美的商品卡片展示
- 🏷️ **商品標籤**：顯示限量版、預購等特殊標籤

## 技術棧

- **React 18** - 前端框架
- **TypeScript** - 型別安全
- **Material-UI (MUI)** - UI 組件庫
- **Emotion** - CSS-in-JS 樣式解決方案

## 專案結構

```
src/
├── components/          # React 組件
│   ├── Header.tsx      # 頂部導航欄
│   ├── Sidebar.tsx     # 側邊欄
│   ├── ProductCard.tsx # 商品卡片
│   └── ProductGrid.tsx # 商品網格
├── data/               # 資料層
│   └── products.ts     # 商品資料和邏輯
├── types/              # TypeScript 型別定義
│   └── index.ts
├── App.tsx             # 主應用程式組件
└── index.tsx           # 應用程式入口點
```

## 安裝與運行

1. **安裝依賴套件**
   ```bash
   npm install
   ```

2. **啟動開發伺服器**
   ```bash
   npm start
   ```

3. **開啟瀏覽器**
   訪問 [http://localhost:3000](http://localhost:3000)

## 主要功能說明

### 導航系統
- **ARTIST**：按娛樂公司分類，可展開查看旗下團體
- **CATEGORY**：按商品分類篩選
- **EVENT** 和 **STORY**：預留位置（未實作）

### 商品篩選
- 娛樂公司篩選（HYBE、YG 等）
- 團體篩選（BTS、BLACKPINK 等）
- 分類篩選（專輯、服飾、應援棒等）

### 商品排序
- 新商品
- 低價位 / 高價位
- 人氣商品
- 試用評價
- 高點擊數

### 商品展示
- 商品圖片
- 商品名稱和描述
- 價格和評分
- 銷售數量
- 特殊標籤（限量版、預購）

## 設計特色

- **簡潔現代**：採用極簡設計風格
- **高質感**：精心設計的卡片和動畫效果
- **使用者友善**：直觀的導航和篩選系統
- **視覺層次**：清晰的資訊架構和視覺引導

## 注意事項

- 目前使用模擬資料，實際應用中需要連接後端 API
- 商品圖片使用 `/img` 資料夾中的隨機圖片
- Logo 使用 `/logo.png`
- 購物車和結帳功能尚未實作
