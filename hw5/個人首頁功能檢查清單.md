# 個人首頁功能檢查清單

## ✅ 已實作的所有功能

### 📍 Header 區域（置頂區域）
- [x] **左箭頭按鈕** - 返回上一頁
- [x] **姓名** - 從 OAuth 提供商取得（例如：ziyenhua b12705022）
- [x] **貼文數量** - 動態顯示 "X posts"

### 🖼️ 背景圖區域
- [x] **背景圖** - 由用戶上傳（預設灰色）
- [x] **Edit Profile 按鈕** - 位於背景圖右下方（只在自己的頁面顯示）
  - 半透明白色背景
  - 毛玻璃效果 (`backdrop-blur-sm`)
  - Hover 時完全不透明

### 👤 個人資料區域
- [x] **大頭貼（Avatar）**
  - 尺寸：128x128 像素
  - 圓形顯示
  - 白色邊框（4px）
  - 位置：中間對齊背景圖底部（`-mt-16`）
  - 如果沒有圖片，顯示姓名首字母

- [x] **Follow 按鈕**（只在別人的頁面顯示）
  - 位於右上方
  - 未追蹤：黑底白字 "Follow"
  - 已追蹤：白底黑字 "Following"
  - Hover 已追蹤狀態：紅色半透明背景，顯示 "Unfollow"

### 📝 個人資訊
- [x] **姓名** - 從 OAuth 取得（粗體，大字）
- [x] **@UserID** - 註冊時設定的 UserID（灰色）
- [x] **個人簡介（Bio）** - 用戶自訂（最多 160 字元）
- [x] **Following 數量** - 追蹤中的用戶數
- [x] **Followers 數量** - 粉絲數

### 📑 標籤頁
- [x] **Posts 標籤** - 顯示用戶的所有貼文和轉發
  - 公開給所有人查看
  - 包含原創貼文
  - 包含轉發的貼文（顯示「XXX 轉發了」）
  - 按時間排序（最新到最舊）

- [x] **Likes 標籤** - 顯示用戶按讚過的貼文
  - 只有本人可見（別人看不到）
  - 按時間排序（最新到最舊）

## 🔄 互動效果

### Edit Profile 按鈕
- 位置：背景圖右下方
- 樣式：半透明白色背景 + 毛玻璃效果
- Hover：完全不透明
- 點擊：開啟編輯個人資料 Modal

### Follow 按鈕（三種狀態）

#### 狀態 1: 未追蹤
- 顯示：`Follow`
- 樣式：黑底白字
- Hover：深灰色背景

#### 狀態 2: 已追蹤（正常）
- 顯示：`Following`
- 樣式：白底黑字 + 灰色邊框

#### 狀態 3: 已追蹤（Hover）
- 顯示：`Unfollow`
- 樣式：紅色半透明背景 + 紅色文字 + 紅色邊框
- 提示：點擊將取消追蹤

## 📊 資料來源

### 從 OAuth 提供商取得（註冊時）
- ✅ **姓名**（name）- Google/GitHub/Facebook 帳號的顯示名稱
- ✅ **Email** - 用於帳號識別
- ✅ **大頭貼** - OAuth 提供商的頭像 URL

### 用戶註冊時設定
- ✅ **UserID** - 3-20 個字元，只能包含字母、數字和底線

### 用戶自行編輯
- ✅ **Bio（個人簡介）** - 最多 160 字元
- ✅ **背景圖（Cover Image）** - 待實作圖片上傳功能
- ✅ **大頭貼更新** - 待實作圖片上傳功能

### 系統動態計算
- ✅ **Posts 數量** - 用戶發表和轉發的貼文總數
- ✅ **Following 數量** - 追蹤的用戶數
- ✅ **Followers 數量** - 粉絲數
