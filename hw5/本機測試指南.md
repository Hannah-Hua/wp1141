## 🐛 常見問題排解

### 問題 1: MongoDB 連線失敗

**錯誤訊息：**
```
MongoServerError: bad auth : authentication failed
```

**解決方法：**
1. 確認密碼正確（沒有特殊字元問題）
2. 確認 Network Access 已設定 0.0.0.0/0
3. 確認用戶已建立在正確的資料庫

---

### 問題 2: Google OAuth 錯誤

**錯誤訊息：**
```
redirect_uri_mismatch
```

**解決方法：**
1. 檢查 `.env.local` 的 `NEXTAUTH_URL=http://localhost:3000`
2. 在 Google Cloud Console 確認 callback URL：
   - `http://localhost:3000/api/auth/callback/google`
3. 重新啟動開發伺服器

---

### 問題 3: 頁面空白或一直 Loading

**可能原因：**
- MongoDB 沒連上
- 環境變數沒設定

**解決方法：**
1. 打開瀏覽器開發者工具（F12）查看 Console 錯誤
2. 檢查終端機的錯誤訊息
3. 確認 `.env.local` 所有必要變數都有填

---

### 問題 4: 無法儲存貼文

**檢查項目：**
1. MongoDB 連線是否正常
2. 是否已完成註冊流程
3. 瀏覽器 Console 是否有錯誤
