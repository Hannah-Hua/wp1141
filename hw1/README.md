

## 🚀 如何開啟網頁

### 方法一：直接開啟 HTML 檔案
1. 下載專案檔案
2. 雙擊 `index.html` 檔案
3. 網頁會在預設瀏覽器中開啟

### 方法二：使用本地伺服器（推薦）
```bash
# 1. 安裝依賴套件
yarn install

# 2. 編譯 TypeScript
yarn build

# 3. 啟動伺服器
yarn serve

# 4. 開啟瀏覽器訪問
# http://localhost:8000
```
