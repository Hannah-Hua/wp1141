# 使用說明 - 階段1 功能測試

## 🎯 階段1 功能驗收清單

### 1. 登入/註冊功能
- [ ] 訪問 http://localhost:5173 應該自動導向登入頁面
- [ ] 點擊「註冊」連結可以切換到註冊頁面
- [ ] 在註冊頁面輸入任意資料可以成功註冊並自動登入
- [ ] 在登入頁面輸入任意 email 和密碼可以成功登入

### 2. 主頁面 - 雙視圖
- [ ] 成功登入後應該看到主頁面
- [ ] 左側顯示地圖視圖（目前是示意圖）
- [ ] 右側顯示咖啡廳列表
- [ ] 可以切換「地圖」、「雙視圖」、「列表」三種模式
- [ ] 顯示當前登入使用者名稱
- [ ] 有「+ 新增咖啡廳」、「📝 我的到訪」、「♥ 願望清單」按鈕

### 3. 咖啡廳列表功能
- [ ] 顯示預設的 5 間咖啡廳
- [ ] 每間咖啡廳顯示：名稱、類別、評分、價格、描述、地址、設施（WiFi、插座）
- [ ] 搜尋框可以根據名稱或地址搜尋
- [ ] 類別篩選按鈕可以篩選不同類別的咖啡廳
- [ ] 點擊咖啡廳卡片會導向詳情頁面
- [ ] 選中的咖啡廳會有高亮邊框

### 4. 咖啡廳詳情頁面
- [ ] 顯示完整的咖啡廳資訊
- [ ] 有「♡ 加入願望」或「♥ 已收藏」按鈕
- [ ] 有「✏️ 編輯」按鈕
- [ ] 有「🗑️ 刪除」按鈕
- [ ] 有「📝 記錄到訪」按鈕
- [ ] 點擊「記錄到訪」會展開表單
- [ ] 可以選擇日期、評分、填寫備註
- [ ] 送出後顯示成功訊息

### 5. 新增/編輯咖啡廳
- [ ] 點擊「+ 新增咖啡廳」進入新增表單
- [ ] 所有必填欄位有 * 標記
- [ ] 名稱、描述、地址欄位為必填
- [ ] 座標必須在有效範圍內（緯度 -90~90，經度 -180~180）
- [ ] 可以選擇類別（下拉選單）
- [ ] 可以設定評分（0-5）
- [ ] 可以設定價格等級（1-4）
- [ ] 可以勾選設施（WiFi、插座）
- [ ] 驗證失敗顯示錯誤訊息
- [ ] 送出後回到首頁並顯示新咖啡廳
- [ ] 在詳情頁點擊「編輯」可以修改現有咖啡廳
- [ ] 編輯表單預填現有資料

### 6. 到訪記錄頁面
- [ ] 點擊「📝 我的到訪」進入到訪記錄頁面
- [ ] 顯示所有到訪記錄（預設有 2 筆）
- [ ] 每筆記錄顯示：咖啡廳名稱、日期、評分、備註
- [ ] 記錄按日期排序（最新在前）
- [ ] 點擊咖啡廳名稱可以導向詳情頁
- [ ] 每筆記錄有刪除按鈕（🗑️）
- [ ] 刪除前會確認
- [ ] 底部顯示統計資訊：
  - 總到訪次數
  - 不同咖啡廳數量
  - 平均評分

### 7. 願望清單頁面
- [ ] 點擊「♥ 願望清單」進入願望清單頁面
- [ ] 顯示所有收藏的咖啡廳（預設有 2 間）
- [ ] 每間咖啡廳顯示：名稱、類別、地址、評分、設施、備註
- [ ] 有「查看詳情」按鈕
- [ ] 有「移除」按鈕
- [ ] 移除前會確認
- [ ] 底部顯示統計：想去的咖啡廳數量

### 8. 導覽與返回
- [ ] 每個子頁面都有「← 返回」按鈕
- [ ] 返回按鈕會回到首頁
- [ ] 頁面切換流暢無錯誤
- [ ] 瀏覽器的上一頁/下一頁按鈕正常運作

### 9. 登出功能
- [ ] 點擊「登出」按鈕會登出
- [ ] 登出後回到登入頁面
- [ ] 登出後無法訪問受保護頁面（會重新導向登入頁）

### 10. UI/UX
- [ ] 所有頁面美觀且配色一致
- [ ] 按鈕 hover 效果清晰
- [ ] 表單欄位 focus 有視覺回饋
- [ ] 錯誤訊息清楚明顯（紅色）
- [ ] 成功訊息有適當提示
- [ ] 載入狀態處理得當
- [ ] 沒有明顯的 UI bug

## 🧪 測試流程建議

### 完整測試流程（約 5-10 分鐘）

1. **首次進入**
   - 開啟瀏覽器訪問 http://localhost:5173
   - 應該看到登入頁面

2. **註冊新帳號**
   - 點擊「註冊」連結
   - 輸入：
     - 使用者名稱：`測試使用者`
     - Email：`test@example.com`
     - 密碼：`password123`
     - 確認密碼：`password123`
   - 點擊「註冊」
   - 應該自動登入並進入主頁面

3. **瀏覽咖啡廳**
   - 查看預設的 5 間咖啡廳
   - 測試切換「地圖」、「雙視圖」、「列表」模式
   - 在搜尋框輸入「Nomad」測試搜尋
   - 點擊類別篩選測試篩選功能
   - 點擊任一咖啡廳進入詳情頁

4. **咖啡廳詳情與互動**
   - 查看咖啡廳完整資訊
   - 點擊「♡ 加入願望」測試收藏功能
   - 點擊「📝 記錄到訪」
   - 填寫表單並送出
   - 確認顯示成功訊息

5. **新增咖啡廳**
   - 回到首頁
   - 點擊「+ 新增咖啡廳」
   - 填寫表單：
     - 名稱：`測試咖啡廳`
     - 類別：`辦公友善`
     - 描述：`這是一間測試咖啡廳`
     - 地址：`台北市信義區市府路 1 號`
     - 緯度：`25.0375`
     - 經度：`121.5644`
     - 評分：`4.5`
     - 價格等級：`$$`
     - 勾選 WiFi 和插座
   - 點擊「新增」
   - 確認回到首頁並看到新增的咖啡廳

6. **編輯咖啡廳**
   - 點擊剛新增的咖啡廳
   - 點擊「✏️ 編輯」
   - 修改名稱為「測試咖啡廳（已修改）」
   - 點擊「更新」
   - 確認修改成功

7. **刪除咖啡廳**
   - 在同一個詳情頁點擊「🗑️ 刪除」
   - 確認刪除
   - 回到首頁確認已刪除

8. **查看到訪記錄**
   - 點擊「📝 我的到訪」
   - 查看預設的 2 筆記錄 + 剛剛新增的記錄
   - 查看統計資訊
   - 點擊其中一筆記錄的咖啡廳名稱
   - 確認能正確導向

9. **查看願望清單**
   - 回到首頁
   - 點擊「♥ 願望清單」
   - 查看收藏的咖啡廳
   - 點擊「查看詳情」測試導覽
   - 點擊「移除」測試刪除功能

10. **登出測試**
    - 回到首頁
    - 點擊「登出」
    - 確認回到登入頁面
    - 嘗試直接訪問 http://localhost:5173/visits
    - 確認被重新導向到登入頁面

## ⚠️ 已知限制（階段1）

1. **假資料**：所有資料只存在於記憶體中，重新整理頁面會重置
2. **假登入**：任何 email 和密碼都可以登入
3. **地圖視圖**：目前只是示意圖，階段3會整合真實的 Google Maps
4. **無資料持久化**：關閉視窗或重新整理會遺失所有更改
5. **無使用者權限**：沒有真正的使用者權限控管
6. **無後端驗證**：所有驗證只在前端進行

## 📝 注意事項

- 確保前端開發伺服器正在運行（`npm run dev`）
- 使用 Chrome、Firefox 或 Safari 等現代瀏覽器
- 建議使用無痕模式避免快取問題
- 如果遇到問題，嘗試清除瀏覽器快取或重啟開發伺服器
