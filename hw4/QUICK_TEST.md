# 階段4 快速測試（5分鐘）

## 🚀 前置準備

確保兩個伺服器都在運行：

```bash
# 終端機 1
cd /Users/hannah/wp1141/hw4/backend
npm run dev
# ✅ http://localhost:3000

# 終端機 2  
cd /Users/hannah/wp1141/hw4/frontend
npm run dev
# ✅ http://localhost:5173
```

---

## ✅ 5 分鐘快速測試

### 1️⃣ 註冊與登入（1分鐘）

1. 開啟 http://localhost:5173
2. 點擊「註冊」
3. 填寫任意資料註冊
4. ✅ 應該自動登入並看到主頁面

---

### 2️⃣ 新增咖啡廳（1分鐘）

1. 點擊「+ 新增咖啡廳」
2. 填寫表單（名稱、描述、地址必填）
3. 點擊「新增」
4. ✅ 回到主頁看到新咖啡廳
5. **重新整理頁面**（F5）
6. ✅ 咖啡廳還在（資料已存到後端）

---

### 3️⃣ 記錄到訪（1分鐘）

1. 點擊任一咖啡廳
2. 點擊「📝 記錄到訪」
3. 填寫表單並送出
4. ✅ 顯示成功訊息
5. 點擊「📝 我的到訪」
6. ✅ 看到剛才的記錄

---

### 4️⃣ 願望清單（1分鐘）

1. 回到主頁，點擊任一咖啡廳
2. 點擊「♡ 加入願望」
3. 輸入備註（可選）
4. ✅ 按鈕變成「♥ 已收藏」
5. 點擊「♥ 願望清單」
6. ✅ 看到剛才加入的咖啡廳

---

### 5️⃣ 登出與登入（1分鐘）

1. 點擊「登出」
2. ✅ 回到登入頁
3. 重新登入
4. ✅ 所有資料都還在

---

## 🎯 成功標準

如果以上 5 個步驟都通過：

✅ **階段4 完成！前後端整合成功！**

---

## 🐛 如果有問題

### 問題：無法載入資料

**檢查**：
1. 後端是否運行？`curl http://localhost:3000`
2. 前端是否運行？訪問 `http://localhost:5173`
3. 瀏覽器 Console 是否有錯誤？（F12）

### 問題：登入後看不到咖啡廳

**檢查**：
1. 後端資料庫是否有資料？
   ```bash
   cd backend
   sqlite3 dev.db "SELECT * FROM cafes;"
   ```
2. 檢查瀏覽器 Network 標籤，API 請求是否成功

### 問題：Token 錯誤

**解決**：
1. 打開瀏覽器 Console（F12）
2. 執行：`localStorage.clear()`
3. 重新整理頁面
4. 重新登入

---
