# Doodle Jump 遊戲

一個簡單有趣的跳躍遊戲，控制角色不斷向上跳躍，避開敵人，挑戰最高分數！

## 遊戲玩法

- 使用 **A/D 鍵** 或 **方向鍵** 控制角色左右移動
- 角色會受到重力影響向下掉落
- 碰到平台會自動向上跳躍
- 避開紅色的敵人，碰到就會遊戲結束
- 跳得越高，分數越高

## 如何開始

1. 安裝依賴：
   ```bash
   npm install
   ```

2. 啟動遊戲：
   ```bash
   npm start
   ```

3. 在瀏覽器中打開 http://localhost:3000

## 遊戲規則

- 從螢幕底部掉出 = Game Over
- 碰到紅色敵人 = Game Over  
- 分數根據跳躍高度計算
- 遊戲結束後點擊「重新開始」按鈕繼續挑戰

## 控制說明

- **A 鍵** 或 **← 鍵**：向左移動
- **D 鍵** 或 **→ 鍵**：向右移動
- 角色會自動從螢幕一邊環繞到另一邊
