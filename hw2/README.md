# Doodle Jump 遊戲

這是一個使用 React、TypeScript 和 CSS 製作的 Doodle Jump 風格遊戲。

## 遊戲特色

- 🎮 **固定視窗大小**: 360 × 640 像素的遊戲畫面
- 🏃 **玩家角色**: 彩色圓形角色，會受到重力影響
- 🧱 **隨機平台**: 自動生成的平台供玩家跳躍
- ⌨️ **鍵盤控制**: 使用 A/D 鍵或方向鍵控制左右移動
- 🔄 **水平環繞**: 從螢幕一邊出去會從另一邊出現
- 💥 **碰撞檢測**: 精確的平台碰撞和跳躍機制
- 🎯 **分數系統**: 根據跳躍高度計算分數
- 🔄 **重新開始**: Game Over 後可以重新開始遊戲

## 如何運行

1. 安裝依賴：
   ```bash
   npm install
   ```

2. 啟動開發服務器：
   ```bash
   npm start
   ```

3. 在瀏覽器中打開 http://localhost:3000

## 遊戲控制

- **A 鍵** 或 **左方向鍵**: 向左移動
- **D 鍵** 或 **右方向鍵**: 向右移動
- **空格鍵**: 重新開始遊戲（在 Game Over 畫面）

## 遊戲規則

1. 玩家會受到重力影響不斷下降
2. 碰到平台會自動向上跳躍
3. 從螢幕底部掉出即 Game Over
4. 分數根據跳躍高度計算
5. 跳得越高，會生成更多新平台

## 技術特色

- 使用 React Hooks 進行狀態管理
- TypeScript 提供類型安全
- CSS3 動畫和漸變效果
- 響應式設計，支援不同螢幕尺寸
- 60 FPS 流暢遊戲體驗
